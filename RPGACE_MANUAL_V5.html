Knowledge Architecture
Productorum Totalis — the taxonomy

A Linnaeus-style 24-phylum classification of all music production knowledge. Every piece of learning, every beat, every encyclopedia entry, every Feynman session is tagged to a node. The taxonomy is the shared language of every AI system in RPGACE.

Why a taxonomy and not just notes
Flat notes accumulate. A taxonomy connects. When a Feynman session on Dorian Mode scores 7/10, the taxonomy node knows: studied twice, never applied in a beat, gap score high. That gap score influences the agenda generator. Oracle reads it before teaching. The knowledge pipeline targets it. The beat log checks it. One data point — six downstream effects.

AI Architecture
How the AI systems communicate

At full capacity, no single Claude call happens in isolation. Every AI output is structured data that feeds the next system. The taxonomy is the shared language between them all.

Oracle → Encyclopedia
When Oracle identifies a production concept in conversation, it auto-triggers an encyclopedia entry with phylum assignment. Feynman Loop watches Oracle output for encyclopedia-style headers and offers a mastery session immediately.
Content Intel → Taxonomy
Whisper transcribes a tutorial. Claude extracts structured insights with: concept + why it matters + FL Studio application + phylum node. Minimum 2 sentences. Single-word insights are rejected. Each insight updates its node.
Beat Log → Canva → Four formats
Beat logged → taxonomy node pulled → Phylum XI maps scale to colour → Phylum XXV matches filmmaker → Canva autofill creates cover art → 3000×3000 (Beatstars), 1080×1080 (Instagram), 1280×720 (YouTube), 1080×1920 (Reels).
Feynman → Schedule → Agenda
Feynman session scores a concept 1–10. Score inverted to gap_score. High gap → spaced repetition entry created in rpgace_sched_agendas. Agenda generator picks it up at the correct interval. Do Now button fires the next Feynman session.
Morning Brief → Day shape
n8n 8am: Gmail unread + shift schedule + YouTube stats delta pulled. Claude synthesises sub-400-word brief. Agenda generator reads shift difficulty from the brief and weights today's task complexity accordingly.
Taxonomy → Prod. Oracle prompts
Every Prod. Oracle command is pre-seeded with taxonomy context. Gap Finder reads gap_score DESC. Beat Analysis reads Phyla I–IV nodes. Best Resources reads youtube_educators field. The buttons are not generic prompts — they are taxonomy-aware queries.
Full Capacity
What runs without being asked

When all phases are complete and n8n is running, these workflows fire automatically. The system moves from reactive (you ask, it answers) to proactive (it prepares, you create).

8:00 AM daily
n8n fires Morning Brief. Gmail unread flagged, today's shifts, YouTube stats delta from last week all pulled. Claude writes sub-400 words. Agenda generator reads shift difficulty and weights tasks for the day. Delivered to Dashboard before the first coffee.
6:00 AM daily
Daily Knowledge Pipeline: n8n finds 2 relevant YouTube tutorials based on current taxonomy gaps. Whisper transcribes audio locally. Claude extracts structured insights (concept + why + FL Studio application + phylum node). Encyclopedia updated, taxonomy nodes enriched. 720 knowledge reports per year at steady state.
On beat log
Taxonomy nodes tagged from key, BPM, mood selection. Phylum XI maps scale to colour palette. Phylum XVI provides beat selling angle. Phylum XXV matches filmmaker visual style. Canva autofill generates cover art in 4 formats. Beatstars description written. Neural Frames brief drafted. All without leaving RPGACE.
Monday 7:00 AM
Fourth rota extractor runs via Browser Use (or n8n). Logs into Fourth, navigates to schedule, extracts shifts for the next 3 weeks. Writes to Supabase rpgace_shifts. Calendar updates automatically. No screenshot, no manual paste.
Sunday 6:00 PM
Weekly Review fires. Sessions logged, beats made, insights extracted, knowledge scored. Gap analysis from taxonomy. Monday agenda pre-built. Journal entry saved to Supabase and Notion. Ready before the week starts.
On video drop
Video pipeline detects new .mp4 in watch folder. auto-editor removes silence. Whisper transcribes. Claude generates EDL (keep/cut decisions). RPGACE Video tab shows review UI. Approve → FFmpeg renders. 4 exports: main edit, Shorts, Reels, thumbnail still. DaVinci for colour only.

The mission
What this is really about

RPGACE is not a productivity tool. It is a system that removes every excuse between Alex and the music. The shifts are real. The time is limited. The knowledge has to compound. The content has to be consistent. The system handles everything it can so the human can handle everything it can't.

Non-negotiables
Russian/French/London cultural perspective stays in the voice. Independent, no label, no compromise on creative direction. Music production knowledge must be genuinely mastered, not just accumulated. Every tool in RPGACE must pass the 48-hour test.
Phase 1 target
2 beats/week. 2 YouTube tutorials/week. 2 Instagram posts/week. System used daily. Taxonomy growing. Feynman Loop completing at least one cycle per week.
Phase 2 target
1,000+ YouTube subscribers. 40+ beats on Beatstars. First licensing deal. Cover art automation live. Visual treatment pipeline working end to end.
Phase 4 target
RPGACE is a product. Other independent producers use it. The taxonomy is public and crowdsourced. The system that was built for one producer serves many.
