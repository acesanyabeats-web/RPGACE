Technology
The AI Stack
Every AI in the system and the specific result it produces.
A
Claude (Anthropic) — Sonnet 4.6
The intelligence layer across the entire system. Powers the Oracle advisor, agenda generation, Feynman Loop sessions, content repurposing, Visual Treatment documents, script writing, insight extraction, and every structured output the system produces. Every AI-generated result originates here.
SB
Supabase (PostgreSQL)
Persistent knowledge layer. Stores the production knowledge taxonomy (25 phyla, 700+ planned nodes), journal entries, encyclopedia entries, video pipeline jobs, and content intelligence reports. The long-term memory the system compounds over time.
W
OpenAI Whisper (local)
Transcription engine. Converts any video or audio to timestamped text locally on-device. Powers the Content Intelligence pipeline — competitor tutorials, reference tracks, and own recordings all become searchable, analysable text before Claude processes them.
CO
Composio
Integration middleware. Connects Claude to Gmail, Notion, YouTube Analytics, and Instagram without managing individual OAuth flows. The Oracle can draft emails, save entries to Notion, and pull channel data on command during a conversation.
n8n
n8n (self-hosted automation)
Workflow automation layer. Runs the Daily Knowledge Pipeline (6am daily: finds 2 production tutorials, transcribes, sends to Claude, updates taxonomy automatically). Also handles the Morning Brief, Content ID alerts, and rota extraction. Replaces Zapier — free and self-hosted.
PW
Playwright + Claude Vision
Work rota extraction. Logs into the employer's scheduling portal, takes precision screenshots of the rota grid, and sends them to Claude Vision which extracts every shift date and time into structured data. Eliminates manual schedule entry entirely.
NF
Neural Frames
AI video generation. Takes exported beat stems and a Claude-written Visual Treatment Document (directing brief, mood, filmmaker references) and generates a 4K beat-synced music video automatically. Every video starts with a concept Claude wrote from the beat's production data.
V
Vercel (Edge Runtime)
Hosting and serverless functions. All Claude API calls, Composio integrations, and data operations route through Vercel Edge Functions — meaning the frontend never exposes API keys and every AI call has a controlled, auditable server layer in between.
Features
Core Modules
What the system does and what AI drives each result.
01
Shift-Aware Agenda Engine
Pulls today's work shifts, cross-references free windows, your current knowledge level, and your work style preference — then generates 5 personalised production tasks calibrated to your actual available energy. A heavy bar shift produces lighter cognitive tasks. A rest day produces a full creative session plan.
Claude · Supabase · Composio
02
Feynman Loop — Verified Learning System LIVE
Three-phase AI learning session triggered from anywhere in the app — the Oracle chat, an Encyclopedia entry, or a learning agenda. Phase 1: explain the concept in plain language. Phase 2: Claude identifies the biggest gap and probes it with one question. Phase 3: full session report with a score out of 10, verified knowledge, gaps found, a specific next study action, and a YouTube content angle. On completion: report saved to Journal, taxonomy node updated, spaced repetition agenda auto-created 3 or 7 days out depending on score.
Claude · Supabase
03
Productorum Totalis — Production Taxonomy
A Linnaeus-style hierarchical classification of all music production knowledge across 25 phyla — from melody theory and drum architecture to beat selling strategy and visual film direction. Every concept studied, every beat made, and every tutorial processed feeds back into this tree. Over time it becomes a mapped record of everything learned, applied, and verified — with source links, FL Studio implementations, producer references, and YouTube educator links per node.
Claude · Supabase · n8n
04
Daily Knowledge Pipeline
Automated 6am workflow. Finds the 2 most relevant new production tutorials uploaded in the last 24 hours, transcribes them locally, sends transcripts to Claude for structured analysis, extracts key concepts, maps them to taxonomy nodes, enriches encyclopedia entries, and files a brief in the Morning Dashboard. Running daily for a year produces over 700 structured knowledge reports — a searchable, AI-indexed production library built automatically in the background while working shifts.
n8n · Whisper · Claude · Supabase
05
Content Intelligence — Competitor Analysis
Submit any YouTube URL. The local pipeline downloads it, Whisper transcribes it with word-level timestamps, Claude extracts production techniques, maps them to taxonomy nodes, writes structured insights with FL Studio application notes, and saves everything to the knowledge base. Competitor tutorials become your encyclopedia entries.
Whisper · Claude · Supabase
06
Oracle AI Advisor
The main conversational layer. Text and image input. Knows your current XP level, shift schedule, encyclopedia depth, and verified Feynman knowledge — so advice is calibrated to where you actually are, not a generic response. Can draft Gmail, save to Notion, pull YouTube stats, and trigger any Composio-connected service mid-conversation. The Production Oracle variant operates as a structured 300IQ production teacher using a Socratic question framework.
Claude · Composio · Gmail · Notion · YouTube
07
AI Video Pipeline
Full post-production pipeline for FL Studio tutorial videos. Drop raw footage in — silence is removed automatically, Whisper transcribes, Claude aligns the transcript to the target script and generates an Edit Decision List. The Review tab shows each cut decision. Approve in ~5 minutes. Renders automatically to four formats: YouTube 16:9, Shorts 9:16, Instagram Reels, thumbnail still. Claude also writes the Visual Treatment Document upfront — the directing brief that tells Neural Frames exactly what the video should look and feel like based on the beat's production data.
Claude · Whisper · Neural Frames · n8n
08
Content Repurposing Engine
One YouTube video script becomes four pieces of platform-native content in one Claude call: a 60-second Short, an Instagram Reels caption with hook and hashtags, a 60-word email blurb with subject line, and a Twitter/X thread. Each opens with a different line. Each matches the voice. Built to clear a content backlog in a single session.
Claude · Composio
09
Morning Brief
Dashboard button. Pulls Gmail from the last 12 hours, today's shift schedule, and a scan of new production concepts filed overnight. Returns a structured 3-section brief under 400 words: what needs a reply before the shift, what can wait, and what new knowledge was filed. Reads in 90 seconds. All infrastructure already connected — zero new services required.
Claude · Gmail · Supabase
10
Beat Log + Taxonomy Connection
After every production session: log the beat with key, BPM, primary technique, and which taxonomy nodes were applied. The system marks those nodes as "used in a beat." Over time the Encyclopedia shows a filter: entries never applied in a production session — your actual to-do list for the next session. Beat Log also triggers the content button: one Oracle call generates a YouTube tutorial angle, Instagram Reels concept, title suggestion, and email blurb from the beat's data.
Claude · Supabase
